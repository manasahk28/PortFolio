Ethereum Fraud Detection Using Graph Neural Networks

A comparative study of 8 GNN architectures for phishing node detection on a 2.97M-node Ethereum transaction graph with a 1:2,500 class imbalance ratio.
• Benchmarked GCN, GAT, GraphSAGE, and a novel Hybrid TGNN under homogeneous and heterogeneous graph representations across unified evaluation metrics.
• Constructed heterogeneous graph with 3 semantic node types (wallet, contract, exchange) and 9 edge relation types for richer structural encoding.
• Designed Hybrid TGNN combining Fourier temporal encoding, multi-head self-attention, and GraphSAGE aggregation - achieving AUC-ROC 0.971, Macro F1 0.931.
• Handled extreme class imbalance via equal-ratio sampling and PR-curve-based threshold tuning to maximise phishing recall.
